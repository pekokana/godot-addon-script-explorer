# スクリプトエクスプローラ (Godot エディタプラグイン) Ver1.2

## エディタで現在選択/編集中の GDScript の情報を表示します**:

1. Godot スクリプトエディタで任意のスクリプトを開きます
2. 右側に **「SE」** というラベルの付いたドックが表示されます
3. ドックには以下の情報が表示されます:
- スクリプト名
- クラス/基底クラス
- メソッドリスト
- プロパティリスト
- シグナルリスト

4. メソッドをクリック → カーソルがそのソース行にジャンプします
5. プロパティをクリック → カーソルがそのソース行にジャンプします
6. シグナルをクリック → カーソルがそのソース行にジャンプします
7. ドック内容をフローティングウィンドウに切り替えて表示します。

### フローティングウィンドウの利用方法

ドック右上の ◱ボタン を押すとフローティング画面が表示されます。

フローティング画面を修了させたい場合は、閉じるボタンをクリックします。

### フローティングウィンドウの特徴
- エディタから独立した小窓でスクリプト情報を確認できる
- 好きな場所に配置可能  
- サイズ変更可能  
- ドック版と同じ内容・同じ動作  
- 切り替えはいつでもワンクリック

## インストール
1. `addons/script_explorer` をプロジェクトにコピーします
2. Godot で、「プロジェクト」→「プロジェクト設定」→「プラグイン」→「スクリプトエクスプローラ」を有効化します

## 注記
- Godot 4.x 用に実装されています
- `script.source_code` を正規表現で解析します。本番環境レベルの解析には、GDScriptAnalyzer をご検討ください。
- フローティングウィンドウは EditorPlugin を通じて EditorBaseControl に動的追加し、独立した Window として管理しています
